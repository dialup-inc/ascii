## Installation

**Download the fast way:**

```sh
# Paste this command into your terminal to install
bash <(curl https://dialup.com/ascii)
```

**Or, build from from source:**

```sh
git clone https://github.com/dialup-inc/ascii.git
cd ascii
make
```
