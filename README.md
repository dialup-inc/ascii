## Installation

*The fast way*
```sh
# Paste this command into your terminal to install
bash <(curl https://dialup.com/ascii)
```

*Building from source*
```sh
git clone https://github.com/dialupdotcom/ascii_roulette.git
cd ascii_roulette
make
```
